# 🌦 Weather App

A full-stack Weather Forecast Application built using React.js and Node.js (Express).

This app allows users to search any city and view real-time weather details like temperature, humidity, wind speed, and weather conditions using the OpenWeatherMap API.

It also includes error handling for invalid city inputs and uses a backend server to securely fetch API data.

---

## 🚀 Tech Stack
- React.js
- Node.js
- Express.js
- OpenWeatherMap API
